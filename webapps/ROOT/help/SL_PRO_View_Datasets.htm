Data Management Help

Viewing Data Set Information

What is a Data Set, and what is it used for?

 

  1. On the Welcome screen, select Data Management.
  2. (Optional) Search for a specific Data Set: Enter any search term into the Search field.
  3. Click the name of the Data Set to see information about the sequence data included in the Data Set, as well as QC reports.

 

Notes:

  • To show or hide table columns: Click , then check or uncheck column name(s) in the dialog box that displays.
  • To sort by a table column: Click the column title.

SMRT Analysis

Use SMRT Link's SMRT Analysis module to perform secondary analysis of sequencing data generated by one or more sequencing runs.

  1. You create and start analyses: Specify the data (BAM or RS II format) to be analyzed, as well as which secondary analysis applications to use.
  2. After the analysis has completed, you can view the secondary analysis results.

 

How are collections, samples, cells, and analyses related?

  • Cells or collections and well samples have a one-to-one relationship and can be used more or less interchangeably.

  • Analyses can be performed on data from one or multiple cells or collections, or on a sample for multiplexed experiments.

Run QC

SMRT Link's Run QC module displays information about every run for every Sequel™ System connected to SMRT Link. Use the Run QC module to:

 

  • Review the overall performance of a run using metrics such as Read Length and Productivity.

  • Observe trends by using the histograms of Read Length per cell.

 

To access Run QC: On the Welcome screen, select Run QC.

 

Notes:

  • Use the buttons to filter the list of runs displayed by specifying Complete, Running, Terminated Failed, Aborted, and/or Ready runs. (Note: If you changed the filters, they will reset to the default the next time you access the main Run QC screen.)
  • To sort the table by a field, click the column name.

Default Table Fields

  • Name: A list of all runs for the instruments connected to SMRT Link. Click a run name to view more detailed information on the Individual Run Page.
  • Run Date: The data and time when the run was started.
  • Status: The current status of the run. Can be one of the following: Running, Completed, Failed, Terminated, or Unknown.
  • Summary: A description of the run.
  • Instrument: The name of the instrument.

Additional Table Fields

To show or hide table fields: Click , then check or uncheck column name(s) in the dialog box that displays.

 

  • Instrument SN: The instrument's serial number.
  • Instrument SW: The version of Sequel™ Instrument Control Software (ICS) installed on the instrument.
  • Primary Analysis SW: The version of Primary Analysis software installed on the instrument.
  • Run ID: An internally-generated ID number identifying the run.
  • UUID: Another internally-generated ID number identifying the run.
  • Completed At: The time of run completion.
  • Total Cells: The total number of SMRT Cells used in the run.
  • Completed Cells: The number of SMRT Cells that generated data for the run.
  • Failed Cells: The number of SMRT Cells that failed to generate data during the run.

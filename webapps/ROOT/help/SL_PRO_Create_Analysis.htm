SMRT® Analysis Help

Starting analyses

To start a new analysis

  1. On the Welcome screen, select SMRT Analysis > New Analysis.
  2. Specify whether to use BAM-format data (generated by the Sequel™ System) or data generated on an RS II System. Note: To analyze RS II data, you must first convert it using the Convert RS to BAM application.
  3. In the Select Data box, select one or more sets of data.
  4. Click Add to add the selected sequence data for analysis, then click Next.
  5. Enter a name for the analysis. Note: At this point, you cannot select different data to use for the analysis. To do so, click Cancel and restart the process.
  6. Select a secondary analysis application to use for the analysis. (For information on the applications supplied by Pacific Biosciences, click here.)
    Note: If you selected RS II data, the only application available is Convert RS to BAM. After you convert the data to the BAM file format, you'll be able to analyze the data using all analysis applications.
  7. Review any required parameters, and modify if needed. If the application requires a reference sequence, select one from the Reference list. (The reference sequences in the list were imported into SMRT Analysis.)
  8. (Optional) Click Advanced Analysis Parameters and fill in as necessary. (These parameters are set to default values, and need only be changed when analyzing data generated in non-standard experimental conditions.)
  9. Click Start to submit the analysis.

To start an analysis after viewing sequence data

  1. On the Welcome screen, select SMRT Analysis, then click View Sequence Data.
  2. Click the appropriate button to see information on:
  • Raw Data, BAM: Sequencing data in BAM format; the Sequel™ System generates data in this format.
  • Raw Data, RS II: Sequencing data generated by an RS II instrument.
  • Note: Click the appropriate button to view References or Barcodes.
  1. Click the link for the sequence data.
  2. Click Analyze.
  3. Enter a name for the analysis.
  4. Select a secondary analysis application to use for the analysis. (For information on the applications supplied by Pacific Biosciences, click here.)
    Note: To analyze RS II data, you must first convert it using the Convert RS to BAM application. After you convert the data to the BAM file format, you'll be able to analyze the data using all analysis applications.
  5. If the application requires a reference sequence, select one from the Reference list. (The reference sequences in the list were imported into SMRT Analysis.)
  6. (Optional) Click Advanced Analysis Parameters and fill in as necessary. (These parameters are set to default values, and need only be changed when analyzing data generated in non-standard experimental conditions.)
  7. Click Start to submit the analysis.

Secondary Analysis Applications Provided by Pacific Biosciences

Following are descriptions of the secondary analysis applications provided by Pacific Biosciences. (Click the links for more detailed information.)

 

Assembly (HGAP 4)

  • Use this application to generate de novo assemblies of genomes.

 

Barcoding

  • Use this application to analyze multiplexed samples.

 

CCS Mapping

  • Use this application to generate consensus sequences from single molecules.

 

Circular Consensus Sequences (CCS 2)

  • Use this application to identify consensus sequences for single molecules.

 

Convert BAM to FASTX

  • Use this application to convert sequence data in BAM file format to the FASTX file format.
  • This application has one advanced parameter: Minimum Subread Length - the minimum length of subreads to write out to FASTA/FASTQ files.

 

Convert RS to BAM

  • Use this application to convert sequence data generated on an RS II system in HDF5 file format to the BAM file format, compatible with SMRT Analysis v3.1.0. After you convert the data to the BAM file format, you'll be able to analyze the data using any of the applications.

 

Iso-Seq

  • Use this application to characterize transcripts and splice variants.

 

Iso-Seq Classify Only

  • This is only the Classify step of the Iso-Seq algorithm. Sequencing reads are classified into full length or non-full length reads.

 

Long Amplicon Analysis (LAA 2)

  • Use this application to identify phased consensus sequences from a heterogeneous pool of amplicons.

 

Resequencing

  • Use this application to map sequencing reads against a reference sequence, and to identify variants.

 

Site Acceptance Test (SAT)

  • Use this application to generate a report displaying instrument acceptance test metrics. (The application is designed only for analysis of Site Acceptance data.)

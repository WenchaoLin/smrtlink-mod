Creating a New Run Design

Use SMRT Link's Run Design module to create Run Designs, which then becomes available from Sequelâ„¢ Instrument Control Software (ICS) - the touchscreen software. Users can then load the instrument and start the run.

  • Runs are available for multiple instruments at a given site.

 

  1. On the Welcome screen, select Run Design.
  2. Click New Run Design.
  3. Enter a Run Name. (The software creates a new Run Name based on the current date and time; you can edit the name as needed.)
  4. (Optional): Enter Run Comments, Experiment Name, and Experiment ID as needed. (Experiment ID must be aplhanumeric.)
  5. Next, you specify information for the first sample that will be part of the run. Enter a Sample Name.
  6. (Optional) Enter any Sample Comments.
  7. Specify the well used for this sample.
  8. Specify whether or not to use Magbead Loading. When OFF is selected, a diffusion run will be performed.
  9. (Optional) Specify a DNA Control Complex.
  10. Enter or scan a Template Prep Kit barcode number.
  11. Enter or scan a Binding Kit barcode number.
  12. Enter or scan a Sequencing Chemistry barcode. (This is the barcode number of the Sequel Sequencing Plate.)
  13. Specify an Insert Size, ranging from 500 to 20,000 base pairs.
  14. Specify the collection time per SMRT Cell, ranging from 30 to 360 minutes.
  15. (Optional) Click Advanced Options, then specify the length of time (60, 120 or 240 minutes) for immobilization of SMRTbells using MagBeads.
    Note: The default for diffusion runs is 1 hour, and the default for MagBead runs is 2 hours.
  16. Sample options:
  • Click Create. This starts a new, empty sample.
  • Click Copy. This starts a new sample, using the values you entered in the first sample.
  • Click Delete. This deletes the current sample.
  1. When you have filled in all the samples, click Save - this saves the entire Run Design. The new Run Design displays on the main Run Design page.
  2. (Optional) Click View Summary to view a table summarizing the entire run design.

SMRT Analysis Help

Base Modification and Motif Analysis Application

Use this application to identify putative sites of base modifications, as well as common bacterial base modifications (6-mA, 4-mC, and optionally TET-converted 5-mC), and then analyze the methyltransferase recognition motifs.

  • Filters reads by length and quality, maps them against a provided reference sequence, and then calls variants.

  • Detection can use either a control sample or an in silico control consisting of expected kinetic signals.

  • For details about the underlying algorithm used by this application, click here.

 

Reference Sequence (Required):

  • Specify a reference sequence to align the SMRT Cells reads to so as to produce a consensus sequence.

 Specify whether or not to merge chunked/gathered .bam files
Hit Policy

randombest

 Specify how to treat multiple hits:
  • random: Selects a random hit.
  • all: Selects all hits.
  • allbest: Selects all the best score hits.
  • randombest: Selects a random hit from all best score hits.
  • leftmost: Selects a hit which has the best score and the smallest mapping coordinate in any reference.
Algorithm Options   List of space-separated arguments passed to BLASR. Default: -minMatch 12 -bestn 10 -minPctIdentity 70.0

Viewing secondary analysis results

  1. On the Welcome screen, select SMRT Analysis. You see a list of all analyses.
  2. (Optional) Click the Successful button to see only successfully-completed analyses.

  3. (Optional) Search for a specific analysis: Enter any search term into the Search field.

  4. Click the analysis link of interest.

  5. Click Analysis Overview > Status to see analysis information status, including which application was used for the analysis, and the inputs used.

  6. Click Analysis Overview > Thumbnails or Display All to view thumbnails of the reports generated for the analysis. Click a thumbnail to see a larger image.

  7. Depending on the application used for the run, different analysis-specific reports are available. For mapping applications only: Click Mapping Report > Attributes to see an overall summary of the mapping data.

  8. To view analysis log details, click Data > Analysis Log.

  9. To visualize the secondary analysis results, click the SMRT View button. SMRT View is a genome browser that visualizes sequencing data generated by the Sequel System.

 

SMRT View Notes:

  • The analysis must have completed successfully and produced alignments as output for the SMRT View button to display.
  • To run SMRT View, you must have 64-bit Java 8 installed on your local Windows or Macintosh computer. Click here for details on installing 64-bit Java 8.

 

To download data files created by SMRT Link:
You can use these data files as input for further downstream processing, pass on to collaborators, or upload to public genome sites.

  1. Click Data > File Downloads, then click the appropriate file. The file is downloaded according to your browser settings.

 

Notes:

  • To sort by a table column: Click the column title.

Long Amplicon Analysis (LAA 2) Application

Use this application to determine phased consensus sequences for pooled amplicon data.

 

The LAA 2 application:

  • Allows for accurate allelic phasing and variant calling in large genomic intervals.
  • Supports the analysis of novel haplotypes in biomedical loci of interest, such as the HLA region in the human genome.

  • Can pool more than 5 distinct amplicons. Reads are clustered into high-level groups, then each group is phased and consensus is called using the Arrow algorithm.

  • If the sample is barcoded, can optionally partition reads by barcode.

 

The application includes five main steps:

  1. Coarse clustering: Group reads from different amplicons into different clusters; detect read-read overlaps; build an overlap graph, then cluster the overlap graph to break the graph into the final clusters.

  2. Waterfall: Align additional reads against coarse consensuses from each cluster, adding them to the cluster with the greatest similarity.

  3. Phasing: Load the reads for each cluster into the Arrow consensus software and find an initial consensus. Recursively split reads from different haplotypes or other PCR products based on high scoring mutations evaluated by Arrow.

  4. Consensus: Generate a final consensus for each haplotype or PCR product using Arrow.

  5. Post-Processing Filters: Detect and remove PCR artifacts. Chimeric sequences are identified using the UCHIME algorithm, and other PCR artifacts are identified by overall consensus quality.

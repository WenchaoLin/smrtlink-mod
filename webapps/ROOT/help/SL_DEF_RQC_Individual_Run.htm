Run QC Help

Individual Run Page

Access this screen by clicking a run name.

 

  • Run Date: The date and time when the run was started.
  • Instrument: The name of the instrument.
  • Instrument SN: The serial number of the instrument.
  • Description: The description, as defined when creating the run.
  • Run ID: The ID automatically assigned to this run.

Sample Run Setup Table

  • Well: The ID of an individual well used for this sample.
  • Name: The sample name, as defined when creating the run.
  • Status: The current collection status for the SMRT® Cell. This can be one of the following: Complete, Collecting, Aborted, Failed, In Progress, or Pending.
  • Movie Time (minutes): The length of the movie associated with this SMRT Cell.
  • Total Bases (GB): Calculated by multiplying the number of productive (P1) ZMWs by the mean polymerase read length; displayed in Gigabases.
  • Read Length: Polymerase reads are trimmed to the high quality region and include bases from adapters, as well as potentially multiple passes around a circular template.
  • Polymerase: The mean trimmed read length of all polymerase reads. The value includes bases from adapters as well as multiple passes around a circular template.
  • Insert: The average length of the Read of Insert, which is a representative read of a DNA molecule from a single ZMW; that is, the sequence of a DNA molecule read from a single ZMW. On circularized SMRTbell™  templates that are shorter than the read length, a Read of Insert length distribution will closely resemble the insert size distribution.

  • Productivity

  • P0: The number and percentage of ZMWs that are empty, with no polymerase.

  • P1: The number and percentage of ZMWs that are productive and sequencing.

  • P2: The number and percentage of ZMWs that are not P0 (empty) or P1 (productive). This may occur for a variety of reasons and the sequence data is not usable.

Histograms

Click the > arrow to expand rows to view histograms for each SMRT Cell where data was successfully transferred. These include:

  • Reads of Insert (length)
  • Polymerase reads (length)

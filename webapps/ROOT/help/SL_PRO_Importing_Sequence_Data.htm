Importing Sequence Data

Importing Sequel Sequence Data

  1. On the Welcome screen, select SMRT Analysis.
  2. Click View or Import Sequence Data.
  3. Click Import and choose Sequel Sequence Data.
  4. Navigate to the .subreads.xml file containing information about Sequel-format sequence data, such as paths to the BAM files.
  5. Click Import. The sequence data is imported and becomes available in SMRT Link.

 

Importing Reference Sequence Files

  1. On the Welcome screen, select SMRT Analysis.
  2. Click View or Import Sequence Data.
  3. Click Import and choose References.
  4. Navigate to the .fa or .fasta file containing the reference sequence to import.
  5. Click Import. The reference sequence file is imported and becomes available for use in starting analyses.

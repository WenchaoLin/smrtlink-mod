Run Design Help

Run Design

Use SMRT Link's Run Design module to create, edit, or import Run Designs, which then becomes available from Sequel™ Instrument Control Software (ICS) - the touchscreen software. Users can then load the instrument and start the run.

  • Runs are available for multiple instruments at a given site.

 

Note: Sequel™ ICS allows you to create a quick run, where all cells must use the same type of loading and movie time.

 

Creating New Run Designs

 

Editing or Deleting an Existing Run Design

  1. On the Welcome screen, select Run Design.
  2. Click the name of the Run Design to edit or delete.
  3. (Optional) Click Show Summary to view a table summarizing the entire run design.
  4. (Optional) Click Delete to delete the current Run Design.
  5. Edit any of the fields.
  6. Click Save.

 

Importing Run Designs

  1. On the Welcome screen, select Run Design.
  2. Click Import Run Design.
  3. Select a CSV file containing the run design and click Open. The run becomes available from the instrument.
  • For information on the contents of CSV Run Design files, click here.
  1. To view the Run Design, click the name, then click View Summary to view a table summarizing the entire run design.

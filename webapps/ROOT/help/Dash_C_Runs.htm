Run QC Help

Runs Page

Use this page to view information about multiple runs on a specific Sequel™ System, for a specified time range.

 

Note: All times displayed are in GMT (Greenwich Mean Time), and match the time in the log files.

To specify a time range:

  1. On the Time Range control (A in the image), click the From field and either type in a starting date, or select a starting date from the calendar.
  2. Click the Thru field and either type in an ending date, or select an ending date from the calendar.
  3. Click Update. The Time Bar (B) displays the time range you just specified.
  4. (Optional) In the Time Bar (B), drag to select a smaller time range (C). That time range now displays in the Runs/Movies/Uptime Bar (D).
  5. (Optional) In the Runs/Movies/Uptime Bar (D), use the Shift key to select a yet smaller time range.


 

In the Runs/Cells/Uptime Bar (D):

  • Double-click a cell or uptime to see further details.
  • Double-click a run to view the Individual Run Page for more details about that run.
  • Move the mouse over a run - the run highlights in the table. If you move the mouse over a run in the table, that run is highlighted in the Time Bar (B).

Table Fields

Note: To sort the table by a field, click the up arrow or down arrow next to the field name:

 

  • Run: A list of runs for the instrument over the specified time period. Click a run to view more detailed information.
  • Started @: The time when the run was started.
  • Status: The current status of the run. Can be one of the following: Running, Completed, Failed, Terminated, or ? (status unknown).
  • SMRT® Cells
  • #: The number of SMRT Cells used in the run.
  • Fail: The number of SMRT Cells used in the run that failed.
  • Total Reads: The total number of reads for the run.
  • Polymerase Reads Polymerase reads are trimmed to the high quality region and include bases from adaptors, as well as potentially multiple passes around a circular template.
  • Length: The mean trimmed read length of all polymerase reads. The value includes bases from adaptors as well as multiple passes around a circular template.
  • Quality: The mean single-pass read quality of all polymerase reads.

  • Reads of Insert

  • Length: The average length of the Read of Insert, which is a representative read of a DNA molecule from a single ZMW; that is, the sequence of a DNA molecule read from a single ZMW. On circularized SMRTbell™ templates that are shorter than the read length, a Read of Insert length distribution will closely resemble the insert size distribution.

  • Quality: The estimated average quality of the Read of Insert.

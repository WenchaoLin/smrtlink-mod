SMRT Analysis Help

Barcoding Application

Use this application to analyze multiplexed samples.

  • Barcoded SMRTbell™ template are SMRTbell™ templates with two barcoded adapters, located on both ends of an insert.

  • With symmetric barcodes, each barcode sequence identifies a single bin for demultiplexing reads. With paired barcodes, each unique pair of barcodes should be listed as two sequentially-named FASTA sequences.

  • Uses a FASTA-format file to identify ideal barcode sequences.

 

Barcodes (Required):

  • Specify a barcode sequence file to separate the reads.

Run QC Help

Individual Run Page

This page displays information about one specific run. You access this page from the Runs page by:

  • Double-clicking a run in the Run/Cells/Uptime Bar or
  • Clicking a run link.

 

Notes:

  • To sort a table by a field, click the up arrow or down arrow next to the field name:
  •  All times displayed are in GMT (Greenwich Mean Time), and match the time in the log files.

Run Summary

  • Instrument: The name of the instrument.
  • Started: The time when the run was started.
  • Completed: The time when the run was completed.
  • Status: The current status of the run. This can be one of the following: Running, Completed, Failed, Terminated, or ? (status unknown).
  • Sequencing Kit: The name of the sequencing kit used, including the lot and expiration date. If two plates are used for the run, the lot and expiration are listed for each.
  • Run Description: The description, as defined when creating the run.

Sample Run Setup Table

  • Well: The ID of an individual well used for this sample.
  • Sample Name: The sample name, as defined when creating the run.
  • Template Prep Kit: The version used for this run. (Move the mouse over the field to see the lot number and expiration date.)
  • Binding Kit: The version used for this run. (Move the mouse over the field to see the lot number and expiration date.)
  • DNA Control: The DNA Control Complex used.
  • Loading: <????>
  • Insert Size (bases): The insert size selected when creating the run. These size ranges match those shown in the Template Preparation and Sequencing Guide.
  • Stage Start: Whether or not Stage Start was used. With Stage Start active, the sequencing reaction starts at the Optical stage, just before data acquisition begins. This captures sequence data at the 5' end of the SMRTbell™ template that would otherwise be skipped.
  • Comments: Entered when creating the run.

Sample Results Table

Note: Move the mouse over a row to see information about the cell tray used.

 

  • SMRT Cell: The number of the SMRT Cell associated with the well.
  • Well: The well number associated with this SMRT Cell.
  • Sample Name: The sample name, as defined when creating the run.
  • Started @: The time that the data acquisition started on this SMRT Cell.
  • Status: The current collection status for the SMRT Cell. This can be one of the following: Collecting/Collected, Collection Failed/Aborted, Analyzing, Analysis Failed/Aborted,Transferring/Transferred, Transfer Failed/Aborted.  Note: A status that includes a question mark (?) indicates that the status is older than 1 day.
  • Movie Length (mins): The length of the movie associated with this SMRT Cell.
  • Total Bases (GB): Calculated by multiplying the number of productive (P1) ZMWs by the mean polymerase read length; displayed in Gigabases.
  • Polymerase Reads: Polymerase reads are trimmed to the high quality region and include bases from adaptors, as well as potentially multiple passes around a circular template.
  • Length: The mean trimmed read length of all polymerase reads. The value includes bases from adaptors as well as multiple passes around a circular template.
  • Quality: The mean single-pass read quality of all polymerase reads.

  • Reads of Insert

  • Length: The average length of the Read of Insert, which is a representative read of a DNA molecule from a single ZMW; that is, the sequence of a DNA molecule read from a single ZMW. On circularized SMRTbell™  templates that are shorter than the read length, a Read of Insert length distribution will closely resemble the insert size distribution.

  • Quality: The estimated average quality of the Read of Insert.

  • Control Reads

  • #: The total number of control reads that passed filtering.

  • Length: The mean mapped read length of the polymerase reads from the control sample.

  • Quality: The mean single-pass accuracy of the mapped polymerase reads from the control sample.

  • Template

  • Adapter Dimer: The % of pre-filter ZMWs which have observed inserts of 0-10bp. These are likely adapter dimers.

  • Short Insert: The % of pre-filter ZMWs which have observed inserts of 11-100bp. These are likely short fragment contamination.

  • Productivity

  • Empty (P0): The number and percentage of ZMWs that are empty, with no polymerase.

  • Productive (P1): The number and percentage of ZMWs that are productive and sequencing.

  • Other (P2): The number and percentage of ZMWs that are not P0 (empty) or P1 (productive). This may occur for a variety of reasons and the sequence data is not usable.

  • Local Base Rate:  The average polymerization rate. This only measures the rate during periods when a single active polymerase is gathering data from a ZMW; for example, it ignores pausing or delays due to multiply-loaded wells.

Histograms

Histograms display for each SMRT® Cell where data was successfully transferred. These include:

  • Reads of Insert (length and quality)
  • Polymerase reads (length and quality)
  • Loading evaluation: Unfiltered reads vs polymerase reads (in kilobases), fraction of the bases in HQ regions.

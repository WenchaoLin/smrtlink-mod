SMRT® Link User Roles  <VERIFY/EDIT>

SMRT® Link supports five user roles:

  • PacBio SuperUser

  • Administrator  (Note: Customer IT Administrators are assigned the Administrator role.)
  • Manager
  • Lab Technician
  • Bioinformatics

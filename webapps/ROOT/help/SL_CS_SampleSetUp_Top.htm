Sample Setup (Available Q3 2016)

Use SMRT Link's Sample Setup module to calculate binding and annealing reactions for preparing DNA samples for use on the Sequelâ„¢ System.

You can then print the instructions for use in the lab.

Create a new calculation

  1. On the Welcome screen, click New Calculation.
  2. Enter the sample name.
  3. Enter the sample concentration, in ng/ul.
  4. Enter the available sample volume, in ul.
  5. Enter the insert size, in base pairs.
  6. Specify whether or not to use Size Selection. (This removes unwanted fragments from the mixture based on size, using Ampure PB Beads, manual isolation from an agarose gel, or automated gel isolation.)
  7. Specify whether to use Diffusion or MagBead loading.
  8. Specify whether or not to use a DNA Control Complex.
  9. Enter the number of SMRT Cells to bind.
  10. Do one of the following:
  • Click Copy to start a new sample using the information you entered. You can then quickly edit specific fields for each sample.
  • Click Remove to delete the current calculation.
  • Click +New Sample to a new, empty, sample.
  1. To print the calculation(s), use your browser's Print command (Ctrl-P).

View existing calculation

On the Sample Setup screen, click an existing sample name, then do one of the following:  <VERIFY>

  • Click View to display the calculation in PDF format.
  • Click Download to save the PDF file on your PC.
  • Click Print to print out the calculation for use in the lab.

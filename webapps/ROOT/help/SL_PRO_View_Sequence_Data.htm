Viewing Sequence Data

  1. On the Welcome screen, select SMRT Analysis, then click View or Import Sequence Data.
  2. Click the appropriate button to see information on:
  • Raw Data, BAM: Sequencing data in BAM format; the Sequel System generates data in this format. Click a link for details.
  • Raw Data, RS II: Sequencing data generated by an RS II instrument. Click a link for details
  • References: Reference sequence FASTA files used when creating certain analyses. Click a link for details.
  • Barcodes: Barcodes from barcoded samples. Click a link for details.

 

Note: You can start an analysis directly after viewing BAM or RS II sequence data.

SMRT® Link Help


Use Pacific Biosciences` SMRT Link to:

 

  • Calculate binding and annealing reactions for preparing samples.

  • Create, edit, and import run designs for sequencing runs.

  • View quality metrics for sequencing runs.

  • Manage data generated by sequencing runs.

  • Perform secondary analysis of sequencing data.

 

Sample Setup:

  • Calculate binding and annealing reactions for preparing DNA samples for use on the Sequel™ System. (Available Q3 2016)

 

Run Design:

  • Create instrument run designs. The runs become available from the instrument touchscreen, Sequel™ Instrument Control Software (ICS), so that users can then load the instrument and start the sequencing run.

 

Run QC:

  • Monitor run status and display quality metrics for individual runs.

 

Data Management:

  • Organize data generated by sequencing runs by creating and managing Data Sets.

 

SMRT Analysis:

  • Perform secondary analysis of sequencing data generated by one or more sequencing runs.
